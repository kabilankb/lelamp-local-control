Features
Everything runs locally
No cloud, no Pi, no subscriptions. Your robot, your system, your control.
⚙

5-Axis Servo Control

Control 5 Feetech STS3215 servos via USB serial. Base yaw, base pitch, elbow, wrist roll, and wrist pitch for full articulation.

🎨

40 RGB LEDs

8x5 LED grid for expressive lighting. Terminal simulator on local systems, or real WS281x NeoPixels on Raspberry Pi.

🎤

Voice Interaction

Hands-free conversation with always-on ASR and TTS. Browser Web Speech API or server-side Whisper + Edge TTS.

🤖

AI-Powered Personality

Sarcastic, expressive robot personality powered by Ollama (free), Claude, Gemini, or OpenAI. Your choice of brain.

🎬

Record & Replay

Physically pose the lamp to record movements, then replay them. 11 pre-loaded expressions including nod, excited, shy, and more.

🌐

Web Control Panel

Beautiful browser UI with live chat, LED visualization, color picker, and movement buttons. Open localhost:5000 and go.

Getting Started
Up and running in 4 steps
1

Clone & Install

Clone the repo and run uv sync. All 152 packages install automatically.

2

Connect USB

Plug the Feetech servo controller into your PC via USB. Run lerobot-find-port to detect it.

3

Setup & Calibrate

Assign motor IDs one by one, then calibrate the range of motion for each joint.

4

Launch

Pick your agent — web panel, voice agent, or text chat — and start interacting.

Choose Your Agent
4 ways to interact
Pick the agent that fits your setup. Mix and match freely.
Free

Web Control Panel

Ollama + Browser ASR/TTS

Full web UI with hands-free voice chat, LED visualization, color picker, and movement buttons. Just speak — no buttons needed.

uv run python web_agent.py
Free

Offline Voice Agent

Ollama + Whisper + Edge TTS

Terminal-based voice agent. Speak into your mic, LeLamp responds with movement and speech. 100% offline capable.

uv run python voice_agent.py
API Key

Claude Agent

Anthropic Claude Sonnet

Text chat powered by Claude with tool use. The most capable conversational experience with precise motor and LED control.

uv run python claude_agent.py
API Key

Gemini Agent

Google Gemini 2.0 Flash

Text chat powered by Gemini with tool use. Free tier available with usage limits.

uv run python gemini_agent.py
